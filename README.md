# class-room-web
=============

Requires Node 0.11


## Set up development environment
```
$ npm install
$ bower install
$ gulp compile-scss
```

## Run app
```
$ npm start
```

## Run app in watch mode
```
$ gulp watch
```

## Run app as production
```
$ gulp compile-scss
$ gulp minify
$ NODE_ENV=production npm start
```

## Run Tests
```
$ npm test
```
